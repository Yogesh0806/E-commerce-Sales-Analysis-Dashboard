🛒 Yadav's E-Commerce Sales Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze e-commerce sales performance across different states, categories, products, customers, and payment methods.

The dashboard helps businesses monitor sales trends, profitability, customer purchasing behavior, and product performance to support data-driven decision-making.

🎯 Objectives
Analyze overall sales and profit performance.
Identify top-performing states and product categories.
Understand customer purchasing behavior.
Evaluate payment method preferences.
Track product-level profitability.
Generate actionable business insights.

📂 Dataset Information

The dashboard uses two datasets:

Orders.csv

Contains customer and order information.

Column Name	Description
Order ID	Unique order identifier
Order Date	Date of order
CustomerName	Customer name
State	Customer state
City	Customer city
Details.csv

Contains transaction-level details.

Column Name	Description
Order ID	Unique order identifier
Amount	Sales amount
Profit	Profit generated
Quantity	Quantity sold
Category	Product category
Sub-Category	Product sub-category
PaymentMode	Payment method used
